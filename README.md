# AI聊天助手

一个基于Spring Boot和Vue 3的AI聊天助手项目。

## 项目结构

- `frontend/` - 基于Vue 3的前端界面
- `backend/` - 基于Spring Boot的后端服务

## 快速开始

### 后端

1. 确保已安装Java 17和Maven
2. 进入后端目录：`cd backend`
3. 运行应用：`mvn spring-boot:run`

### 前端

1. 确保已安装Node.js
2. 进入前端目录：`cd frontend`
3. 安装依赖：`npm install`
4. 运行开发服务器：`npm run dev`
